# Autodesk Programming Assignment

This repository contains solutions for the programming assignment.

## Assignment 1
Reverse each word in a string while preserving punctuation and spacing.

Implementation:
- Language: C++
- File: `assignment1/reverse_words.cpp`

The solution iterates through the string and reverses contiguous alphanumeric sequences.

To run:

```bash
g++ reverse_words.cpp -o reverse_words
./reverse_words
